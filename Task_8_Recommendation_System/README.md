# Task 8: AI-Based Recommendation System

This project implements a content-based movie recommendation system using machine learning.

## Approach
- TF-IDF vectorization on movie genres
- Cosine similarity to find similar movies
- Streamlit used for the user interface

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## How to Run
pip install -r requirements.txt  
streamlit run app.py
